# 📱 Courses Application (Jetpack Compose)

A simple and modern **Android Dashboard App** built using **Jetpack Compose**.
This project demonstrates key UI concepts like scrolling layouts, top app bar, menus, and card-based design — perfect for beginners and academic projects.

## 🚀 Features

✨ Clean and minimal UI
📌 Top App Bar with menu (Profile, Settings, Logout)
📜 Vertically scrollable dashboard
➡️ Horizontally scrollable topics (LazyRow)
📦 Course cards with title, description, and status
🔔 Toast messages on user interaction



## 🛠️ Tech Stack

* **Kotlin**
* **Jetpack Compose**
* **Material 3 UI**
* Android Studio

## 📸 UI Overview

### 🔹 Dashboard Screen

* Top App Bar with menu options
* Horizontal topic list (Android, Java, AI, etc.)
* Vertical course cards section

## ⚙️ How It Works

* **Scaffold** → Base layout structure
* **SmallTopAppBar** → Displays title and menu
* **LazyRow** → Shows horizontal topics
* **Column + verticalScroll()** → Enables vertical scrolling
* **Card** → Displays course information
* **DropdownMenu** → Handles menu actions
